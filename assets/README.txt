# Asset Files README

This directory stores templates, images, fonts, and other resource files.

These files are NOT loaded into context, but used in outputs.

Common asset types:
- Templates: .pptx, .docx, project template directories
- Images: .png, .jpg, .svg
- Fonts: .ttf, .woff2
- Data: .csv, .json

Replace this file with actual assets or delete this directory.
